Описание

picColorRange.js - специальный скрипт, который берёт градиент из двух цветов и возвращает промежуточный цвет, в зависимости от положения целевого объекта на градиенте. Такой подход избавляет нас от потребности использовать маски, расширяя возможности для визуализации данных

Применение

Скрипт простой и достаточно его подключить к вашему HTML-документу и вызвать, когда необходимо применять его свойства. Скрипт принимает и отдаёт только RGB-цвета.

<script src="picColorRange.js"></script> // Визуально функция выглядит так
var color = RangeColor( 'rgb(255,0,0)', 'rgb(0,255,0)', 50 );
console.log(color) //"rgb(127,128,0)"
