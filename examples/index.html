Описание

PicColorRange.js - скрипт, который принимает в качестве аргумент два цвета градиента, положение объекта в процентах без значка "%" и возвращает промежуточный цвет, в зависимости от положения объекта. Такой подход избавляет нас от потребности использовать маски, расширяя возможности для визуализации данных

Применение

Скрипт прост в использовании, досататочно подключить к HTML-документу и вызвать функцию там, где необходимо применить свойства скрипта. Скрипт принимает и отдаёт только RGB-цвета

<script src="picColorRange.js"></script> // Визуально функция выглядит так
var color = RangeColor( 'rgb(255,0,0)', 'rgb(0,255,0)', 50 );
console.log(color) //"rgb(127,128,0)"
